# 24-01 - Oracle Bind Transparent Rewriter

<objective>
Accept Oracle-legal user bind names containing `$` or `#`, rewrite only driver-facing SQL placeholders and `sql.Named` keys to go-ora-compatible names, reverse-map errors to original names, and preserve all Phase 22/22.5 reserved-name and audit contracts.
</objective>

<action>
Create `dbee/adapters/oracle_bind_rewrite.go` with the tokenizer, name rewriter, and error reverse mapper.

Required helper shape:

1. `rewriteOracleBindNameForDriver(name string) (driverName string, changed bool, err error)`.
2. `rewriteOracleSQLBinds(query string) (rewritten string, mapping oracleBindRewriteMap, err error)` or equivalent struct.
3. The mapping tracks at least:
   - original user name
   - rewritten driver name
   - `driverToUser map[string]string`
   - `userToDriver map[string]string`
4. Rewriter only changes bind references in code regions matched by `:([A-Za-z_$#][A-Za-z0-9_$#]*)`.
5. Rewriter skips:
   - single-quoted strings with doubled quote escapes
   - q-quoted strings with paired delimiters `[ ]`, `{ }`, `( )`, `< >`, and same-delimiter forms
   - double-quoted identifiers
   - line comments
   - block comments
6. `:=` must not match because `=` is not a valid first bind-name character.
7. Error reverse mapper replaces rewritten names with originals by descending rewritten-name length and wraps with `Unwrap() error`.
</action>
