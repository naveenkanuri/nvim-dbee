---Format microseconds into adaptive human-readable duration.
---@param us number microseconds
---@return string
local function format_duration(us)
  local seconds = us / 1000000
  if seconds >= 60 then
    local minutes = math.floor(seconds / 60)
    local remaining = seconds - (minutes * 60)
    return string.format("%dm %ds", minutes, math.floor(remaining))
  elseif seconds >= 1 then
    return string.format("%.2fs", seconds)
  else
    return string.format("%dms", math.floor(seconds * 1000))
  end
end
```

This intersection avoids false positives from collapsed-but-cached connections: those may still be in cache, but they will NOT trigger `connection_get_structure_async` on this refresh cycle because they are not expanded.

refresh = function()

  -- cached AND currently expanded.

  self._manual_refresh_conns = {}
  for conn_id, _ in pairs(self.structure_cache) do
    if exp[conn_id] then
      self._manual_refresh_conns[conn_id] = true
    end
  end

  self:refresh()
end,
```

**Part C: Update on_structure_loaded** (~line 163):

Add manual refresh detection. When a structure_loaded event fires for a conn_id in `_manual_refresh_conns`, notify the user and remove that conn_id from the set. Once all expected conn_ids have been drained, the set becomes empty and no more notifications fire. This prevents leaking into later auto-load events:

```lua
function DrawerUI:on_structure_loaded(data)
  if not data or not data.conn_id then
    return
  end

  -- Check if this conn_id was part of a manual refresh cycle
  if self._manual_refresh_conns[data.conn_id] then
    -- Remove from expected set (drain approach -- prevents leak into later auto-loads)
    self._manual_refresh_conns[data.conn_id] = nil
    -- Notify on success
    if not data.error then
      local ok_params, conn_params = pcall(self.handler.connection_get_params, self.handler, data.conn_id)
      local name = (ok_params and conn_params and conn_params.name) or data.conn_id
      utils.log("info", "Schema loaded: " .. name)
    end
  end

  self.structure_cache[data.conn_id] = {
    structures = data.structures,
    error = data.error,
  }

  self:refresh()
end
```

This approach:
- Tracks ONLY connection IDs whose structures will be re-fetched (`cached ∩ expanded`)
- Drains the set as responses arrive (removes each conn_id after processing)
- Empty set = no pending manual refresh = auto-load events are silent
- No boolean flag that could persist and leak into unrelated lazy_children expand events
- Non-expanded connections are never added, so later lazy_children expansion is always silent
- connection_get_params wrapped with pcall for robustness, falls back to conn_id
