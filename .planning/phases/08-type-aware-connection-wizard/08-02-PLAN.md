<interfaces>
Wizard open contract:
```lua
local wizard = require("dbee.ui.connection_wizard")

wizard.open({
  relative_winid = self.winid,
  mode = "add" | "edit",
  title = "Add Connection" | "Edit Connection",
  source_meta = { id = "connections.json", name = "connections.json" },
  seed = {
    params = { name = "", type = "postgres", url = "" },
    wizard = nil, -- or persisted metadata from 08-01 helper
  },
  on_submit = function(submission) end,
  on_cancel = function() end,
})
```

Wizard submission contract:
```lua
{
  params = {
    name = "Payroll Prod",
    type = "postgres",
    url = "postgres://user:pass@host:5432/payroll?sslmode=require",
  },
  wizard = {
    db_kind = "postgres",
    mode = "postgres_form",
    fields = {
      host = "host",
      port = "5432",
      database = "payroll",
      username = "user",
      password = "pass",
      sslmode = "require",
    },
    rendered_url = "postgres://user:pass@host:5432/payroll?sslmode=require",
  },
}
```

Lossless fallback rules:
```lua
seed with persisted wizard metadata first
else parse existing params only when the parse is lossless
else fall back to a raw mode:
  - postgres URL for unsupported postgres query params
  - oracle custom JDBC raw text when descriptor cannot be safely decomposed
  - Other raw for non-scoped types
```
</interfaces>
